Công Cụ Xuất File
Tải bản gốc hoặc bản làm rối mã để phân phối cho CLB
📄
Bản Gốc (Code rõ ràng)
Dùng để chỉnh sửa, phát triển. Không chia sẻ ra ngoài.
🔒
Bản Làm Rối (Obfuscated)
JavaScript đã được mã hóa. Dùng để chia sẻ cho CLB sử dụng.
⏳ Quá trình tạo mất khoảng 10–20 giây, vui lòng chờ.
💡 Hướng dẫn: Mỗi lần cập nhật tính năng mới → nhấn Tải Bản Gốc để lưu về máy/chỉnh sửa → nhấn Tạo & Tải Bản Làm Rối để lấy file chia sẻ cho CLB.
✅ Hoạt động hoàn toàn trên máy tính — không cần server, chỉ cần kết nối internet lần đầu để tải thư viện (~2MB). Sau đó có thể upload lên GitHub bình thường.
